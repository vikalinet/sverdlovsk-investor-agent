# 🏭 Помощник инвестора в Свердловской области

AI-агент для поддержки инвестиционной деятельности в Свердловской области.

## Возможности

- 🔍 **Поиск лучших отраслевых практик** в разных регионах РФ
- 📊 **Анализ инвестиционного потенциала** (объекты, виды бизнеса)
- 💰 **Подбор мер господдержки** (гранты, субсидии, фонды)
- 📄 **Подготовка пакета документов** для подачи заявок
- ✅ **Верификация документов** перед подачей

## Структура проекта

```
investor_agent/
├── config/          # Конфигурация
├── src/             # Исходный код модулей
├── templates/       # Шаблоны документов
├── tests/           # Тесты
└── main.py          # Точка входа
```

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Модули

| Модуль | Описание |
|--------|----------|
| `search_module` | Поиск практик и инвестиционных возможностей |
| `database_module` | Работа с БД через MCP-серверы |
| `documents_module` | Генерация и валидация документов |
| `analysis_module` | Анализ и формирование предложений |
